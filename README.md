# js-neural-network-edu

Interactive neural network visualization and training tool built with JavaScript.
Live at: 

## 📌 About the Project
This project is a simple and clear educational tool designed for those who want to learn the basic working principles of artificial neural networks.  
Input, hidden, and output layers are visually drawn on canvas, and the training process can be followed step by step.

## ✨ Features
- Visualization of input, hidden, and output layers
- Training (train) and testing (test) modes
- Observe weight updates and the learning process
- Review training and test results through tables
- LocalStorage support for saving and loading weights

## 🚀 How to Use
- Run the files on a web server or simply open `index.html` directly in your browser.  
- Enter the Input and Target values.  
- Click the **Train** button to train the network.  
- Click the **Test** button to observe the result.  

## 📂 File Structure
- `index.html` → Interface and basic structure  
- `function.js` → Neural network calculations and visualization functions  
- `stylesheet.css` → Modern and professional design styles  

## 🎯 Purpose
This repository provides an interactive example for students and enthusiasts who want to learn about neural networks.  
It offers a visual and experimental learning experience without diving into complex mathematical details.
